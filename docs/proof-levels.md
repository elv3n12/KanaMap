# Niveaux de preuve

Les signalements ne doivent jamais être présentés comme des vérités définitives. Le niveau de preuve indique le degré de documentation disponible.

## L1_TESTIMONY

Témoignage simple, déclaratif, sans pièce jointe.

Label public : `À vérifier`.

## L2_VISUAL

Photo d’emballage, capture d’écran, ticket, étiquette ou vitrine.

Label public : `Documenté partiellement`.

## L3_COMMERCIAL_DOC

Fiche produit, publicité, page e-commerce ou argumentaire marketing.

Label public : `Documentation commerciale`.

## L4_LAB_ANALYSIS

Résultat de laboratoire, chromatographie, test produit ou rapport associatif.

Label public : `Analyse disponible`.

## L5_SANITARY_SIGNAL

Effet indésirable grave, passage médical, déclaration sanitaire, rapport institutionnel ou cas solidement documenté.

Label public : `Signal sanitaire associé`.
